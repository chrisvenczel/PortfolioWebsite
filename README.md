# Portfolio Website

This is the code for my personal portfolio website. It is built using static HTML files generated from a dynamic PHP-based site.

It is deployed for free using [Netlify](https://www.netlify.com/), which serves the static files.

## Prerequisites

- [HTTrack](https://www.httrack.com/) is used for copying the website to static HTML files.
  - Install: `sudo apt-get install httrack`
- PHP 8.0+ CLI is used to serve the website locally.
  - Install: `sudo apt-get install php8.0-cli`
- Node 18+ (for npm scripts)
